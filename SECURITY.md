# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately rather than opening a public issue:

- Preferred: open a [GitHub Security Advisory](../../security/advisories/new)
  for this repository (Security tab → Advisories → Report a vulnerability).
- Alternative: email **TODO: add a contact email here**.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs, configs, or proof-of-concept code

Please don't disclose the issue publicly until it has been addressed.

## Supported Versions

This repository tracks the `main` branch only; there are no maintained
release branches at this time.

## Scope

This repository contains infrastructure-as-code (Docker Compose
configuration and deployment scripts) for running Wazuh. Vulnerabilities in
Wazuh itself should be reported to the
[Wazuh project](https://github.com/wazuh/wazuh/security) directly, not here.
